# DevHead Case Studies

This repository contains public case studies and technical writeups for selected full stack, mobile, e-commerce, API, and deployment projects.

Most of my active development work is in private or organization repositories because it supports real products, client work, and business systems. This repository is used to share project summaries, architecture decisions, technical challenges, and professional proof without exposing private source code.
